# Analyzing_Grammys_Website_Performance_Excel
## Project Background ##
The GRAMMY Awards, established by the Recording Academy in 1959, recognize outstanding achievements in the music industry by honoring artists, songwriters, composers, producers, and other music professionals. In the mid-1990s, the Recording Academy launched GRAMMY.com as the official digital platform for the awards and the organization.

Over the past three decades, the website has evolved into a comprehensive resource featuring award history, nominee and winner databases, news, educational content, and multimedia. This project evaluates the website's usability, information architecture, and user experience to determine whether the current combined platform should remain unified or be separated into two distinct websites: GRAMMY.com for award-related content and RecordingAcademy.com for organizational resources and initiatives.

This project uses user research and website analysis to assess how effectively the current platform meets visitor needs and provides recommendations for improving the site's structure and overall user experience.

Insights and Recommendations are provided on the following key areas:
- **Web Analytics:** Evaluation of the visitor analytics of Grammy.com and recordingacademy.com after the split
- **Viewership Analysis:** An analysis of view patterns for both websites, focusing on bounce rate, pages per session, and average time on site to determine if the split has positively or negatively impacted engagement.
- **Mobile Optimization:** An assessment of engagement from mobile device users and if optimization is necessary
